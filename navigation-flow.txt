# Flow POS Interactive Demo Navigation Flow Configuration
# ============================================================
# This file defines the navigation flow for the interactive demo.
# Format: Each screen definition starts with "SCREEN:" followed by screen data
# Clickable areas are defined with "CLICKABLE_AREA:" lines
# 
# Screen Format: SCREEN:SCREEN_ID|IMAGE_PATH|SCREEN_TITLE|DESCRIPTION
# Clickable Area Format: CLICKABLE_AREA:X_PERCENT|Y_PERCENT|WIDTH_PERCENT|HEIGHT_PERCENT|NEXT_SCREEN_ID|LABEL
# 
# Coordinates are percentage-based (0-100) for responsive design
# Lines starting with # are comments and ignored
# Empty lines are ignored
# ============================================================


# 1. UNDERSTANDING THE COORDINATE SYSTEM:
#    - X_PERCENT: Horizontal position (0 = left edge, 100 = right edge)
#    - Y_PERCENT: Vertical position (0 = top edge, 100 = bottom edge)
#    - WIDTH_PERCENT: Button width as percentage of image width
#    - HEIGHT_PERCENT: Button height as percentage of image height
